This repository has been archived and is no longer actively maintained.

**Reason for Archiving**: This project has been deprecated in favor of [tsup](https://github.com/egoist/tsup), which provides a more efficient and flexible solution for bundling TypeScript projects.

For future development or similar use cases, we highly recommend using [tsup](https://github.com/egoist/tsup).
